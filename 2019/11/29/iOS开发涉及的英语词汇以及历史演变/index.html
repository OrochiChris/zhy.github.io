iOS开发涉及的英语词汇以及学习历程

发表于 2019-11-29 | 分类于 心得

选我所爱,爱我所选，努力成为一名终身学习的程序员

建议大家利用零碎的时间来学习这些英语单词，例如：起床、睡前，坐地铁、公交，饭后，无聊时等等时间，都可以充分利用起来去学习英语，以便将来可以更爽快的阅读开发文档 -> iOS 苹果官方文档

学习方法

1.每天拿起手机时记7个单词，反反复复，温故而知新

2.我经常逛的网站有 Cocoa China，Stack Overflow ， GitHub ，掘金网等

规划

1.iOS开发我个人认为以后的趋势是混合的，并不一定纯原生，特别是flutter和swiftUI，对于不同层次的iOS程序员成长路线和发展方向，我觉得可以分为以下几个方面的内容：

  • 独立App开发
  • 业务能手，业务逻辑抽象
  • SDK 功能组件开发
  • 跨端技术 Weex，React——Native，vue.js
  • 底层研究，iOS编绘，性能，网络，安全等研究。
  • 端上机器学习 Core ML g 和 ARKit 这些技术很有发展前景

2.iOS程序员并非要局限于iOS开发本身，条条大河终入海，iOS程序员要以iOS开发为入口深入下去，不断深入计算机领域，努力走在计算机科学技术发展的前端。

3.虽然系统和iPhone的创新上不及当年，但我相信iOS的发展和它的生态。

4.在选择职业道路的时候其实也可以划分成2种，这也和App store的生态有关：

  • 一个是UI线，在大业务中专门负责业务页面的搭建，沉淀UI组件，好的交互，好的性能优化和画面渲染。
  • 一个是基础架构线，主要实现网络，高可用，App架构等

5.优秀的开发者博客，也是前进的动力：喵神
